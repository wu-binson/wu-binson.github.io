Probability Space

Definition

A probability space or a probability triple $(\Omega, \mathcal F, P)$:

  1. A sample space, $\Omega$, which is the set of all possible outcomes.
  2. A set of events $\mathcal F$, which each event is a set containing zero or more outcomes: $\mathcal F\subseteq2^\Omega$— a set of subsets of $\Omega$.
  3. The Assignment of Probability to the events; that is a function $P$ from events to probability:$P:\mathcal F \mapsto [0,1]$
