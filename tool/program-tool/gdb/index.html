.gdbinit脚本

cgdb

gdb以及自带的gdb GUI非常简陋，这时我们可以使用cgdb， 相当于给gdb穿上了一套装备。在gdb基础上除了有高亮的源代码窗口，可视化的断点，常用功能的键盘快捷键，搜索源代码功能，还有文件对话窗口等功能。

上面设置了搜索大小写不敏感，tab 大小，分屏方式（默认改为左右分屏），搜索高亮，以及加了一个 F9 的快捷键，用于跳出循环。

其他功能可以查看cgdb的官网。

gdbgui

与gdb和cgdb相比，gdbgui是用python开发的基于浏览器的gdb前端，很漂亮，直接用鼠标点就行了，不用记快捷键。

但个人觉得暂时还是cgdb的效率更高，就没怎么使用gdbgui，但gdbgui还在快速发展中，未来应该会更强大。
