Tmux

Tmux是一个用于在终端窗口中运行多个终端会话的工具，即终端复用软件。 它的基础概念和安装方法请看这篇博文, 这里就不赘述了。

下面是tmux的效果图： 如图所示，当前窗口(window)有三个窗格(Pane)，从图下方看出有12个窗口(window)，每个窗口都有自己的名称。这些窗口和窗格之间均可以通过快捷键 快速切换。这些窗口合在一起称为会话(session)，用户可以新建多个会话，并且可以自由切换。
